# Helium Browser Extensions

Privacy and productivity extensions for Helium Browser (Chromium-based).

> **Note**: Helium comes with uBlock Origin pre-installed.

## Installation

Visit the Chrome Web Store links below and click "Add to Chrome/Helium".

## Privacy Extensions

### ClearURLs
**Extension ID**: `lckanjgmijmafbedllaakclkaicjfmnk`
**Chrome Web Store**: https://chromewebstore.google.com/detail/clearurls/lckanjgmijmafbedllaakclkaicjfmnk

Removes tracking elements from URLs.

### Privacy Badger
**Extension ID**: `pkehgijcmpdhfbdbbnkijodmdjhbjlgp`
**Chrome Web Store**: https://chromewebstore.google.com/detail/privacy-badger/pkehgijcmpdhfbdbbnkijodmdjhbjlgp

Blocks invisible trackers automatically.

## AI Extensions

### Claude for Chrome (Official)
**Chrome Web Store**: https://claude.ai/chrome

> **Requirements**: Claude Max plan ($200/month)
>
> Official Anthropic extension that allows Claude to:
> - Read and navigate websites
> - Click buttons and fill forms
> - Maintain context from active browser sessions
> - Take actions on your behalf with permission

**Alternative**: For non-Max users, use https://claude.ai in a browser tab.

### Page Assist (Local AI)
**Extension ID**: `jfgfiigpkhlkbnfnbobbkinehhfdhndo`
**Chrome Web Store**: https://chromewebstore.google.com/detail/page-assist-a-web-ui-for/jfgfiigpkhlkbnfnbobbkinehhfdhndo

Open-source local AI assistant (works with Ollama).
