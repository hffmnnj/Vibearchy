# Wallpaper Credits

Vibearchy includes curated wallpapers from these amazing open-source collections.
All wallpapers are downloaded at install time from their original repositories.

## Nordic Wallpapers

Beautiful wallpapers optimized for the Nord color scheme.

- **Repository:** https://github.com/linuxdotexe/nordic-wallpapers
- **Author:** Sai Nivas Mangu (linuxdotexe)
- **License:** MIT
- **Used in:** Minimal/Clean, Nature/Space categories

## Aesthetic Wallpapers

A curated archive of high-quality aesthetic wallpapers for Linux rices.

- **Repository:** https://github.com/D3Ext/aesthetic-wallpapers
- **Author:** D3Ext
- **License:** MIT
- **Used in:** Riced/Anime, Nature/Space categories

## Catppuccin Wallpapers

Wallpapers matching the Catppuccin Mocha color palette.

- **Repository:** https://github.com/orangci/walls-catppuccin-mocha
- **Author:** orangci
- **License:** Not specified (community collection)

## Contributing

If you'd like to suggest additional wallpaper packs, please:

1. Ensure they have a permissive license (MIT, GPL, CC, etc.)
2. Open an issue on the Vibearchy repository
3. Include the repository URL and license information

## Removing Wallpapers

To remove downloaded wallpapers:

```bash
rm -rf ~/.config/hypr/walls/*
```

To download fresh wallpapers:

```bash
~/Documents/Vibearchy/scripts/wallpapers.sh
```
